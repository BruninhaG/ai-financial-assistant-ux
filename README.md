# Assistente Virtual Financeiro com IA Generativa

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um Assistente Virtual Inteligente focado no setor financeiro. O objetivo principal foi criar uma interface de conversação que não apenas responde a comandos, mas que entende o contexto do usuário, explica produtos complexos e realiza simulações financeiras de forma clara, segura e personalizada, aplicando conceitos sólidos de **UX Design**.

## 🛠️ Funcionalidades Principais
* **Compreensão de Linguagem Natural (NLP):** Interação fluida e contextualizada com o usuário utilizando IA Generativa.
* **Persistência de Contexto:** Capacidade de lembrar o histórico da conversa atual para manter o diálogo coerente.
* **Cálculos Financeiros Demonstrativos:** Ferramenta integrada para simular rendimentos ou parcelamentos básicos.
* **FAQ Inteligente:** Explicação simplificada de produtos e serviços financeiros bancários.

## 🧰 Tecnologias Utilizadas
* **Linguagem:** Python
* **IA Generativa:** (Ex: OpenAI API / Google Gemini API) para o motor do chatbot.
* **Interface / Prototipagem:** (Ex: Streamlit para a interface web rápida ou CLI estruturado).
* **Processamento de Dados:** Pandas para estruturar o conhecimento base.

## 📐 Pilares de UX Aplicados
* **Clareza e Transparência:** Respostas diretas, evitando termos técnicos excessivos (economês) sem explicação.
* **Segurança Emocional:** Tom de voz empático, prestativo e focado em ajudar o usuário a tomar decisões informadas.
* **Prevenção de Erros:** Respostas padronizadas para inputs inválidos ou perguntas fora do escopo financeiro.

## 🚀 Como Executar o Projeto
1. Instale as dependências necessárias: `pip install -r requirements.txt`
2. Configure sua chave de API nas variáveis de ambiente.
3. Inicie a aplicação: (Ex: `streamlit run app.py` ou `python main.py`)
